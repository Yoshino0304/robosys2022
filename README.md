# plusコマンド
入力した値を足し算するプログラムです.

# 動作手順
* 1 Ubuntu 22.04.1 LTSをインストールする.
* 2 plusと同じ中身のファイルを作成する.
* 3 数字のみのファイルを作成する. 
* 4 ./plus < <数字のみのファイル名>で実行する.　

# 必要なソフトウェア
* Python 3.7～3.10

# 動作確認済み環境
* Ubuntu 22.04.1LTS
